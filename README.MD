#Cerinte
```
 1. Scrie un program care sa afiseze pe linie numerele de la 1 la 70 cu mici exceptii:
	  - daca numarul este divizibil cu 3 sa se afiseze "Ding";
	  - daca numarul este divizibil cu 5 sa se afiseze "Dong";
	  - daca numarul este divizibil cu 3 si cu 5 sa se afiseze "DingDong";
  

 2. Scrie o functie de PHP care sa converteasca orice string in urmatorul format: prima litera al fiecarui cuvant sa fie litera mare, iar toate celelalte sa fie cu litera mica.
De exemplu myStringConvert ('un string aleator') sa intoarca: Un String Aleator.
  
 3. Scrie o functie care primeste ca parametru un numar de cel putin 4 cifre si afiseaza inversul.
De exemplu pentru numarul 1234 se va afisa 4321.
 
 4. Sa presupunem ca trebuie sa implementezi o clasa numita Ball. Aceasta clasa trebuie sa aiba un atribut numit ballCount (care incepe de la 0) si o metoda FoundBall. Atunci cand se apeleaza FoundBall, valoarea ballCount este marita cu 1. Daca ballCount este egala cu 7, atunci mesajul "Ai gasit mingea" este afisat, iar ballCount este resetat la 0. 
Cum ati implementa aceasta clasa?
```
